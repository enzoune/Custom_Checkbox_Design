# 🔘 Custom Animated Checkbox (UI Component)

> Um componente de interface moderno e minimalista desenvolvido com HTML5 e CSS3, focado na personalização de elementos de formulário nativos e micro-interações.

## 📖 Sobre o Projeto

Este projeto integra o meu portfólio de estudos em **Programação Full-Stack**. O objetivo foi criar um **Checkbox Customizado** que foge do padrão convencional dos navegadores, oferecendo uma experiência visual mais rica e alinhada a interfaces de usuário (UI) modernas.

O foco do desenvolvimento foi manter a estrutura o mais limpa possível, utilizando técnicas avançadas de CSS para gerenciar estados visuais sem depender de bibliotecas externas ou JavaScript.

## ✨ Destaques e Aprendizados Técnicos

* **Customização de Elementos Nativos:** Técnica de ocultação do `input` original para substituição por uma interface estilizada via `label`, garantindo que o componente continue funcional e acessível.
* **Gerenciamento de Estado Declarativo:** Uso do seletor `:checked` para disparar transições visuais, demonstrando domínio em lógica de interface puramente via CSS.
* **Acessibilidade (A11y):** Uso correto do atributo `for` vinculado ao `id` do input, permitindo que a interação ocorra em toda a área da label, melhorando a usabilidade em dispositivos móveis.
* **Código Limpo e Modular:** Estrutura simplificada em uma única classe `.checkbox`, facilitando a portabilidade do componente para qualquer formulário ou sistema de design.
* **Organização Profissional:** Separação estrita entre a marcação estrutural (`index.html`) e a identidade visual localizada no diretório `css/style.css`.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação do componente de formulário.
* **CSS3:** Estilização, animações de transição e gerenciamento de estados visuais.
